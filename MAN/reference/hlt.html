HLT - Halt CPU





	Usage:	 HLT
	Modifies flags: None


	Halts CPU until RESET line is activated, NMI or maskable interrupt
	received.  The CPU becomes dormant but retains the current CS:IP
	for later restart.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	none		  2	2     5	    4		  1


	- see Instruction Timing
