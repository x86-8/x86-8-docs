ESC - Escape





	Usage:	ESC	immed,src
	Modifies flags: None


	Provides access to the data bus for other resident processors.
	The CPU treats it as a NOP but places memory operand on bus.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	immed,reg	  2   9-20    ?			  2
	immed,mem	  2   9-20    ?			 2-4


	- see Instruction Timing
