LTR - Load Task Register   (286+ privileged)





	Usage:	LTR	src
	Modifies flags: None


	Loads the current task register with the value specified in "src".


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	reg16		  -	17    23    20		  3
	mem16		  -	19    27    20		  5
