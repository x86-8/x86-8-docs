SIDT - Store Interrupt Descriptor Table (286+ privileged)





	Usage:	SIDT	dest
	Modifies flags: none


	Stores the Interrupt Descriptor Table (IDT) Register into the
	specified operand.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	mem64		  -	12    9	    10		  5


	- see Instruction Timing
