JC - Jump on Carry





	Usage:	JC	label
	Modifies flags: None


	Causes execution to branch to "label" if the Carry Flag is set.
	Functionally similar to JB and JNAE.  Unsigned comparision.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	label:	jump	  16   7+m   7+m    3		 2-4
		no jump	  4	3     3	    1


	- see Instruction Timing
