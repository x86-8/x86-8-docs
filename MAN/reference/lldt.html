LLDT - Load Local Descriptor Table  (286+ privileged)





	Usage:	LLDT	src
	Modifies flags: None


	Loads a value from an operand into the Local Descriptor Table 
	Register (LDTR).


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	reg16		  -	17    20    11		  3
	mem16		  -	19    24    11		  5


	- see Instruction Timing
