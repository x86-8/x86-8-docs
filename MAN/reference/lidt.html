LIDT - Load Interrupt Descriptor Table  (286+ privileged)





	Usage:	LIDT	src
	Modifies flags: None


	Loads a value from an operand into the Interrupt Descriptor Table
	(IDT) register.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	mem64		  -	12    11    11		  5


	- see Instruction Timing
