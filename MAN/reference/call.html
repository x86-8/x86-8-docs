CALL - Procedure Call




	Usage:	CALL	destination
	Modifies flags: None

	Pushes Instruction Pointer (and Code Segment for far calls) onto
	stack and loads Instruction Pointer with the address of proc-name.
	Code continues with execution at CS:IP.
