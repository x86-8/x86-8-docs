LAHF - Load Register AH From Flags





	Usage:	LAHF
	Modifies flags: None

	Copies bits 0-7 of the flags register into AH.	This includes flags
	AF, CF, PF, SF and ZF other bits are undefined.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	none		  4	2     2	    3		  1
