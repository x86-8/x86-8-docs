SETO - Set if Overflow (386+ only)





	Usage:	SETO	dest
	Modifies flags: none


	Sets the byte in the operand to 1 if the Overflow Flag is set,
	otherwise sets the operand to 0.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	reg8		  -	-     4	    3		  3
	mem8		  -	-     5	    4		  3


	- see Instruction Timing
