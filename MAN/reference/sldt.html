SLDT - Store Local Descriptor Table (286+ privileged)





	Usage:	SLDT	dest
	Modifies flags: none


	Stores the Local Descriptor Table (LDT) Register into the
	specified operand.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	reg16		  -	2     2	    2		  3
	mem16		  -	2     2	    3		  5


	- see Instruction Timing
