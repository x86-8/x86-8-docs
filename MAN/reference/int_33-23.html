INT 33,23 - Get Language Number




	AX = 23h


	on return:
	BX = language number (with /L switch value):
	   = 0	English     n/a
	   = 1	French	    F
	   = 2	Dutch	    NL
	   = 3	German	    D
	   = 4	Swedish     S
	   = 5	Finnish     SF
	   = 6	Spanish     E
	   = 7	Portuguese  P
	   = 8	Italian     I

	- only works with international version of the mouse driver
	- returns English (0) if not international version

	- see	INT 33,22
